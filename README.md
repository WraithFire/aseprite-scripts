# Aseprite Scripts

A collection of **custom scripts** for [Aseprite](https://www.aseprite.org/) to speed up your pixel art workflow, automate tasks, and enhance creativity.

## Requirements

-   Aseprite v1.2.25 or higher (some scripts may require newer versions)

## Installation

1. Download the script files (`.lua`) you want to use
2. Open Aseprite
3. Go to `File > Scripts > Open Scripts Folder`
4. Copy the downloaded `.lua` files into this folder
5. Restart Aseprite or go to `File > Scripts > Rescan Scripts Folder`

## Usage

1. Go to `File > Scripts`
2. Select the script you want to run
3. Follow any on-screen prompts or dialogs

**Keyboard Shortcuts (Optional)**

You can assign keyboard shortcuts to frequently used scripts:

1. Go to `Edit > Keyboard Shortcuts`
2. Search for your script name
3. Assign a shortcut key combination
